Title                   : Idumea
Filename                : idumea.wad
Release date            : 06/10/2024 (RC1)
Author                  : taviow
Email Address           : [redacted]
Other Files By Author   : Some maps in community projects like Hardfest 2 and Doom 2 in City Only, also some maps in other projects like Plutonic Visions, Spectrum Descent, Waffle House and Nostalgia Zero.
Misc. Author Info       : nothing to point out
Description             : A very large and non-linear MBF21-compatible (-cl 21) city map based around slaughter combat. It's not meant to be super hard slaughter and I placed many more power-ups than you might actually need during your playthrough, but it's hard enough that I've seen it give people some problems.

Most importantly: this release includes a version of the map for low-end computers! You can select which version you want to play in the episode selection screen, or else just warp to 31 to play the low-spec version.

Original playtesters: Insane_gazebo, eater29, finnks13, myolden, maribo, General Roasterock and Jinxie. Thanks to myolden also for creating the Doom 2 in City Only project, this map wouldn't exist without it.

* Construction *

Base                    : New from scratch
Build Time              : 3 frenzied weeks followed by a month of implementing the secret exit (I burned myself out a bit). Several months later, about 3 more weeks trying to work out how to make it run well and implementing the required changes.
Editor(s) used          : Ultimate doom builder, SLADE
Known Bugs              : It is very important to note that the original version of the map will not perform well on anything other than dsda-doom on OpenGL rendering, as people have let me know many times. The map is very big. If it does not perform well please try MAP31, that one should work with any MBF21 compatible port even while using software rendering.
May Not Run With        : Anything besides MBF21 compatible ports, gameplay mods have not been tested.
Tested With             : dsda-doom 0.27.0+ (-cl 21)
