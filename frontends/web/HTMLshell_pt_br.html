Descrição dos Menus

  • jogar: Inicia uma nova campanha ou seleciona a fase de início.
  • carregar: Restaura o progresso salvo (disponível apenas durante a sessão atual do navegador após salvar no menu; perdido ao recarregar a página).
  • som fx: Liga/desliga os efeitos sonoros.
  • musica: Liga/desliga a música.

Controles & Teclas (Keybindings)

  • Mover: Setas direcionais, WASD, teclado numérico ou teclas Q/E (passos laterais)
  • Olhar / Girar Câmera: Mover o mouse (Clique Direito para alternar olhar vertical para cima/baixo) ou usar Setas Esquerda/Direita
  • Atirar / Confirmar: Clique Esquerdo do Mouse, [J], [ENTER] ou [CTRL]
  • Mover Lateralmente (Strafe) / Cancelar: [LSHIFT] / [K]
  • Pular: [ESPAÇO]
  • Mapa: [TAB] ou [BACKSPACE]
  • Menu: [ESC]
  • Mudar de Arma: [F], [O], [P], [X], [Y], [Z] ou roda do mouse
